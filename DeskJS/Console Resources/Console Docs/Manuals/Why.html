Why Now?

So, why now?

With the increasing numbers of JavaScript programmers rising almost everyday, there's a need for such people to have an ecosystem around them designed for their own productivity and skills-growth. The Web Console in Firefox's browser is one such tool. But sadly enough, the browser seems to be the only primary tool for many users and developers alike. Not much has been done in this side of the "programmer's world". Other beginning programmers have also been asking experienced programmers, especially in forums such as Stackoverflow if there actually exists any apps (majorly for Windows) that let them just type in JS (JavaScript) code and quickly execute it, but almost all the answers are tied to using the browser (which is quite good, but one that is very confined or limited), JScript and NodeJS. Except only one did I see a programmer mention Mancy, which is another wonderful JavaScript console built on Web Technologies, but very heavy in terms of memory consumption as reported by some users. Fortunately enough, many have survived with these tools over the years, but for how long?

DeskJS aims to fill a need in this space especially now with the rising numbers of JavaScript programmers. Its portability lets you run JavaScript in any modern Windows Operating System today, giving you the essentials of being productive as a new/experienced programmer. Python (a programming language) which has an amazing following and forum has a really capable console with a lot in stock, but JavaScript seems to still live inside the browser. Thankfully enough, this fact is now changing with such tools as NodeJS, Mozilla's Rhino JavaScript Interpreter, Babel, and JScript. This was one of my inspirational points which was to be a part of this change and goodwill initiative. One that is not confined in competition but in helping change people's lives with regards to the tools that they use everyday. As I always say, it's all about the users!

Furthermore, DeskJS will not just settle for JavaScript's standard features alone... there's definitely more to come.
