JS Objects supported

What's an Object?

An Object is simply a collection of properties and methods put together in one nice, little package that can be stored in memory.

DeskJS supports some of the most relevant and important objects in the JavaScript programming language. They include the Math, Date, RegExp, screen, console, and JSON.

The Math Object

With the Math object, all of the standard properties and methods in JavaScript are well supported, with a few examples including Math.sin(), Math.cos(), Math.PI and Math.tan().

So, for example, you can find the sine of a number like 12 just by writing and executing Math.sin(12) or find the logarithm of the same number by writing and executing Math.log(12). Notice that you wont need to first construct the Object by doing: var x = new Math() like we do with the Date object.

The completions list will also give you all of the supported properties and methods as defined in ECMAScript's 5.1 language specification.

The Date Object

The Date object is also supported with all of its methods. However, you'll need to construct the Object first just like we do in the browser. An example using the Date object is as below:

var d = new Date(); print(d)

However, the Date object is referred to as a static object since as time goes by, the time within the object will not be refreshed to match the current time unless re-declared again (like in the example above). You may therefore be forced to keep on re-declaring a Date object to get the current time.

Also by default, the Date object is not supported in the completions list so you may need to reference the Web to check the methods usable with it.

The RegExp Object

The RegExp or Regular Expressions object lets you perform regular expressions matching in DeskJS.

The same way the Date object is declared is the same way the RegExp object is declared. A very basic example is as shown below:

var re = new RegExp('\\w+');

However, since regular expressions matching is possible by default even for declared strings, you may not always need to do it as in the above way. You can use the search method that comes with strings to perform expression-matching.

To learn more about regular expressions, you can visit Regexone. It's a really nice place to teach yourself on how to use this wonderful programmer's tool. Plus, it's included in most modern programming languages like Python and VisualBasic.NET.

The screen Object

The screen object lets you get information about your computer's screen through JavaScript. Below is the list of properties available in DeskJS's screen object.

width - displays the width of the screen in pixels

height - displays the height of the screen in pixels

availWidth - displays the available width of the screen in pixels

availHeight - displays the available height of the screen in pixels

resolution - displays the full screen's resolution in "W x H" format

colorDepth - this returns the number of bits used to display one color

pixelDepth - this returns the pixel-depth of the screen

The console Object

The console object is by default used to print out information to the console. Though you may still use the print() statement, the console object may hugely benefit those who wish to stick with Industry standards and practices. Just to mention, the console.log() method can print statements while being executed in a loop since it directly logs messages to the console turn-after-turn unlike the print() method which prints messages after a loop has ended. This is a major stronghold of the console.log() method. The console object is supported in the completions list

Below is a list of the methods supported in this object:

console.log() - This method prints messages to the console's results panel

console.count() - This method returns the number of results output in the results panel

console.clear() - This method clears all the results output in the results panel

The JSON Object

Not much can be said about this object but that the basic methods found in the standard JSON object in JavaScript are also supported. Such methods include the JSON.parse() and JSON.stringify(). This object is however not supported in the completions list.
