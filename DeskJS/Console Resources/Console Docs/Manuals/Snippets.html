Snippets

What are snippets?

Snippets are tiny (or even large) chunks of code that are stored and reserved for future use. They can be as large as you want them to be depending on the amount of text you want to save. Snippets are a much easier way of saving your codes rather than using files as they can be easily accessed and understood in DeskJS. Yet still, they are actually files... but very special!

By default, each snippet is stored as an XML (Extended Markup Language) file within the Console Snippets folder, which is accessed from the Console Resources folder. All snippets are saved with the extension ".console-snippet". A snippet can contain a Name, Content (which is your code), and a Description, which lets one know what a particular snippet does when executed. Below is the basic snippet format in DeskJS:

<?xml version="1.0" encoding="UTF-8"?>

<ConsoleSnippet>

    <Description><![CDATA[your description goes here]]></Description>

    <Content><![CDATA[your code goes here]]></Content>

</ConsoleSnippet>

Fortunately enough, you don't need to know anything at all about this format or how to create and save such files as DeskJS lets you plainly create your own snippets without having to define the above format. This was only to give programmers a hands-on approach to manually creating snippets if they so wish to. However for basic users, it's as easy as the press of a key-combination. You can quickly create a new snippet by pressing on CTRL + Q which displays a window that lets you create your snippet with ease. You are be able to navigate through the input-boxes using the ENTER or TAB key. Also, editing and deletion of snippets is possible. To edit your snippets, just press on CTRL + E from the console's editor which displays a search panel to let you quickly search or select the snippet you want to edit. To delete your snippet, just highlight the snippet you want deleted from the snippets list and press on the Delete key. Also, by pressing on Ctrl + W from the console's editor, you will be able to choose any of your snippets and insert them in the console's editor for you to edit and/or execute them.

Also to note, you can still do everything described above without using shortcut keys just by clicking on the snippets icon found in the console's top-left area (which is described as the icons area). Once you hover your cursor over the snippets icon, you will see a tooltip describing the snippets feature.

Snippets are a productivity feature for all kinds of users.
