Scripts

What are scripts?

Scripts, or with regards to DeskJS, JavaScript files, are simply files that contain code, which in our case is JavaScript.

By default, you will be allowed to save all your scripts inside the Console Scripts folder. You may however choose to save them elsewhere in your computer.

Associating JavaScript files with DeskJS

By association, we mean that you are able to allow particular files to be opened with a particular application installed or not installed from your computer. For example, by default, you may find that text files (with a .txt extension type) are opened with Notepad, unless changed.

This means that you can be able to associate all JavaScript files to be launched or executed with DeskJS. This can be achieved by following the guide below:

  • Go to any existing JavaScript file in your computer (they contain a .js extension type). Else if there's none, create one.
  • Right-click on the file and select the properties option at the bottom of the menu that appears. A window will then be displayed containing the properties of the file.
  • Then, click on the Change button right besides the opens with label.
  • Thereafter, you can now browse to the location of DeskJS and choose it. For Windows 8/8.1 and later, to browse to the location of DeskJS, just click on more options, then scroll down the list presented to you and select Look for another app on this PC. You can then scroll to DeskJS's location from there on.

You can still repeat the above process for files of any type in your computer, even for normal Text files (.txt).

If you want to quickly execute scripts, you can still choose to drag-and-drop any script file onto DeskJS or any shortcut that you've created for it. For example, if you've created a shortcut for DeskJS in your Desktop, you can drag-and-drop any script file onto the shortcut and it will be executed right away. This will come in-handy especially for files that contain code for inputs and outputs with the use of popup-boxes such as alert and/or confirm dialogs.
