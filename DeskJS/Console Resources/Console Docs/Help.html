Special commands

You are able to execute some actions or view some resources in the application by typing specialized commands within the console's editor. All you'll need to do is type a few keywords then press on ENTER or F5 to execute them. Also to note, these commands are not case sensitive at all so you can type them in either mixed, upper or lower-case letters.

Below is the list of special commands available:

  • about - lets you view the application's "about" dialog
  • help - (displays this document)
  • docs or manuals - lets you view the application's manuals guide
  • cref - this displays the command-line reference guide
  • sref - this displays the shortcuts reference guide
  • dir - launches DeskJS's current directory in Windows Explorer
  • scripts or sc - launches the scripts directory in Windows Explorer
  • snippets or sn - launches the snippets directory in Windows Explorer
  • themes or th - launches the themes directory in Windows Explorer
  • clear or cls or clr - clears all the console's results
  • exit or quit - exits the application
  • restart - restarts the application

  • start or st - lets you start processes (e.g. applications)

    The start command takes this format: start [process] [arguments]. For example, to start an application such as notepad, you can type start notepad. You can also start uninstalled applications using their full paths such as start c:\users\admin\MyApp\app.exe. You may even include parameters or arguments when launching applications. A very basic example is: start notepad c:\users\admin\Desktop\hello.txt or start notepad++ -help. Please make sure that the process name has no spaces as it will not be started.

  • mkdir or md - (Acronym for Make Directory) lets you easily create directories for placing your scripts.

    The mkdir or md command takes the format: mkdir [foldername]. For example, to create a folder named hello in your Console Scripts folder, you can type mkdir hello or md hello. You can also create folders elsewhere by specifying their new paths, for example mkdir C:/users/admin/Desktop/hello. You may use either backslash or forwardslash characters whe specifying their paths. Spaces can also be included in the folder name (e.g. mkdir hello samples).

  • rmdir or rd - (Acronym for Remove Directory) lets you easily delete or remove any existing directories.

    The rmdir or rd command takes the format: rmdir [foldername]. For example, to delete an existing folder named hello in your Console Scripts folder, you can type rmdir hello or rd hello. You can also delete folders existing elsewhere by specifying their full paths, for example, to delete a folder named hello existing in my Desktop folder, I can simply type rmdir C:/users/admin/Desktop/hello. Additionally, if the folder doesn't exist, you'll be notified on execution. Spaces can also be included in the folder name (e.g. rmdir hello samples).
