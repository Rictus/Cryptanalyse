Shortcuts

Here's a full list of all shortcut keys implemented in DeskJS:

  • Ctrl + O - opens existing script files
  • Ctrl + S - saves your current commands as scripts
  • Ctrl + Q - create new console-snippets
  • Ctrl + W - lists all your existing console-snippets
  • Ctrl + E - edit your existing console-snippets
  • Ctrl + H - shows your commands history list
  • Ctrl + L - lists all the available console themes
  • Ctrl + R - change the editor's pointer character
  • Ctrl + T - enables/disables DeskJS as TopMost window
  • Ctrl + Y - enables/disables saving console-commands
  • Ctrl + U - enables/disables running selections in the console
  • Ctrl + I - change the 'ESCAPE' key's event
  • Ctrl + P - pins the application to the System Tray
  • Ctrl + TAB - shows the quick code-completions list
  • Alt + Plus - increases window transparency level
  • Alt + Minus - reduces window transparency level
  • F5 - alternatively executes JS commands in the console
  • F6 - refreshes the current theme (this shortcut can be used mostly while you're developing/modifying themes)
  • F7 - clears your commands history list (previous commands)
  • F8 - clears all the results output in the results panel
  • F11 - maximizes/restores the console window
  • Up (↑) Arrow key - moves up the list of previous commands
  • Down (↓) Arrow key - moves down the list of current commands
  • ENTER or Return key - executes JS commands in the console
